1. Абсолютная адресация - это тип адресации, при котором адресация происходит относительно всех других адресов. В абсолютной адресации применяются полные адреса, которые указывают на определенный объект в интернете.

2. Относительная адресация - это тип адресации, при котором адресация происходит относительно других адресов. В относительной адресации используются адреса, которые указывают на относительное местоположение определенного объекта.

3. Относительный путь от документа - это путь, используемый при относительной адресации, когда адрес указывает на относительное местоположение объекта относительно документа.

4. Относительный путь от корневого каталога - это путь, используемый при относительной адресации, когда адрес указывает на относительное местоположение объекта относительно корневого каталога.

5. Линейный градиент можно создать, используя свойство CSS «background-image» и значение «linear-gradient». Для создания линейного градиента необходимо указать два или более цветов и направление градиента.

6. Радиальный градиент можно создать, используя свойство CSS «background-image» и значение «radial-gradient». Для создания радиального градиента необходимо указать два или более цветов, центр градиента и радиус градиента.

7. Чтобы создать обтекание изображения текстом слева, необходимо использовать свойство CSS «float». Кроме того, необходимо установить правильные значения для значений «margin» и «padding», чтобы текст и изображение отображались корректно.

8. На вебстраницу можно загружать изображения в форматах JPG, PNG, SVG, GIF и WebP.

9. Свойство background-image используется для задания фонового изображения для элемента. Значения свойства могут быть адресом URL изображения, которое будет использоваться как фоновое изображение, или значениями функции, такими как linear-gradient или radial-gradient, для создания градиента.

10. Размер изображения можно изменить с помощью свойств CSS «width» и «height».

1. Префиксы используются для обеспечения совместимости и для поддержки браузерами новых CSS свойств.


2. Фоновое изображение можно зафиксировать с помощью свойства background-attachment: fixed.


3. Свойство background-blend-mode используется для изменения прозрачности или цвета фонового изображения. Значения свойства включают в себя: normal, multiply, screen, overlay, darken, lighten, color-dodge, color-burn, hard-light, soft-light, difference, exclusion, hue, saturation, color и luminosity.


4. Прозрачность используется для изменения прозрачности или прозрачности цветов элементов HTML.


5. Карта-изображение – это изображение, которое содержит данные в виде изображения и которое предназначено для использования в картах.


6. Чтобы сделать вид маркера графическим, можно использовать свойство background-image, чтобы применить выбранное графическое изображение к маркеру.
