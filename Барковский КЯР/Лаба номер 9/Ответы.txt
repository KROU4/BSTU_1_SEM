1. Модульная сетка - это система размещения элементов, где все элементы выравниваются и разделяются на одинаковые части. Модульная сетка позволяет разработчикам планировать и организовать веб-страницу посредством деления экрана на несколько колонок и строк.

2. Skeleton - это система модульной верстки, которая предоставляет разработчикам мощные инструменты для быстрого и эффективного создания веб-страниц. Skeleton поддерживает две адаптивные модульные сетки, такие как плавающие и grid-верстки.

3. Система Skeleton используется для создания модульной верстки веб-страниц. Для этого необходимо использовать компоненты Skeleton, предоставленные в комплекте с библиотекой. Разработчик может создать адаптивные модульные сетки и использовать их в своем проекте.

4. Стили в файле skeleton.css указывают на стили для настройки модульной сетки. Они помогают разработчику легко и быстро настроить стиль и макет веб-страницы.

5. Основные этапы реализации системы Skeleton: 
• Выбор модульной сетки - выбор типа модульной сетки, который будет использоваться для проекта.
• Настройка модульной сетки - задание значений ширины и высоты для каждой колонки и строки. 
• Настройка элементов - настройка стилей для каждого элемента.
• Проверка кросс-браузерности - проверка корректности отображения на разных браузерах и устройствах.

6. Grid-верстка - это метод для организации веб-страницы с помощью сетки. Он позволяет разместить разные элементы веб-страницы в заранее заданных местах и использовать их для построения адаптивного макета.

7. Свойство grid-template-areas используется для определения имен ячеек для расположения элементов в сетке. Это свойство позволяет разработчикам легко идентифицировать элементы и изменять их положение на странице.

8. Свойство grid-template-columns используется для задания ширины колонок модульной сетки. Это свойство позволяет разработчику настроить размер каждой колонки в сетке.

9. Свойство grid-area используется для задания имени ячейки сетки, к которой может быть привязан элемент. Это позволяет разработчику легко идентифицировать элементы и изменять их положение на странице.

10. Свойство grid-row используется для задания положения элемента в сетке. Это свойство позволяет разработчику настраивать положение элемента в сетке по вертикали.

11. grid-template-columns: 20px 20px 20px; означает, что каждая колонка имеет ширину 20 пикселей.

12. grid-row-end: 4; означает, что элемент заканчивается в 4-й строке сетки.

13. Свойство grid-gap используется для задания расстояния между элементами в сетке. Это свойство позволяет разработчику задать расстояние между элементами и изменять его по мере необходимости.

14. 
<html>
<head>
    <style>
        .container {
            display: grid;
            grid-template-columns: 20px 20px 20px;
            grid-template-rows: 20px 20px 20px;
        }

        .container > div {
            background-color: white;
        }

        .container p {
            background-color: blue;
        }

        .row2 {
            grid-row: 2;
            background-color: red;
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="row1"></div>
        <div class="row2"></div>
        <div class="row3"></div>
        <p>Paragraph</p>
    </div>
</body>
</html>

15. Свойства align-self и justify-self используются для задания выравнивания элементов в сетке. Они позволяют разработчику настроить выравнивание элемента по оси Х и У соответственно.
