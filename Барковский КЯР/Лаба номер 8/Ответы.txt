1. Псевдоклассы :link, :visited, :hover, :active могут использоваться для изменения состояния гиперссылки. 
2. Псевдокласс :hover используется для изменения стилей элемента при наведении мыши на него. 
3. Псевдокласс :active используется для изменения стилей элемента, когда пользователь нажимает на него. 
4. Псевдокласс :link используется для изменения стилей элемента, когда пользователь еще не посещал страницу, содержащую гиперссылку. 
5. Псевдокласс :visited используется для изменения стилей элемента, когда пользователь уже посещал страницу, содержащую гиперссылку. 
6. Вертикальная панель навигации может быть создана с помощью свойства float со значением left или right для элементов панели. 
7. Горизонтальная панель навигации может быть создана с помощью свойства float со значением none и свойства display со значением inline-block для элементов панели. 
8. Выпадающая панель навигации может быть создана с помощью псевдокласса :hover для элементов панели, а также каскадных стилей для вложенных элементов. 
9. Метод CSS-спрайт используется для создания одного большого изображения из нескольких мелких изображений, что позволяет уменьшить количество запросов к серверу. 
11. Ролловер - это элемент пользовательского интерфейса, который позволяет пользователям прокручивать длинные списки или длинные изображения. 
12. Выпадающие элементы могут быть скрыты с помощью псевдокласса :hover и свойства display со значением none для вложенных элементов. 
13. Свойство position используется для управления позицией элемента на странице. 
15. Для выпадающего меню и основного меню необходимо использовать позиционирование со значением relative или absolute.

17. Существует 4 типа позиционирования: относительное, абсолютное, статическое и фиксированное.

18. Относительное позиционирование - это способ расположения элемента относительно его окружающих ближайших элементов.

19. Абсолютное позиционирование - это способ расположения элемента с учетом его положения в отношении предыдущих элементов.

20. Статическое позиционирование - это способ расположения элемента, в котором он остается на месте, независимо от других элементов.

21. Фиксированное позиционирование - это способ расположения элемента, при котором он остается на том же месте при изменении размера окна браузера.
