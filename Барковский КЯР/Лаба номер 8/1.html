CSS Свойства

CSS (Cascading Style Sheets) — формальный язык описания внешнего вида документа, написанного с использованием языка разметки. CSS позволяет определять внешний вид элементов HTML-документа.

В CSS используются различные свойства, которые устанавливают внешний вид элементов. Например, свойство «background-color» задает цвет фона элемента, свойство «font-family» задает семейство шрифтов, а свойство «margin» задает значение отступа.
