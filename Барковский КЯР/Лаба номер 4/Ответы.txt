1. Блочными элементами называются те элементы, которые занимают весь доступный пространство, и имеют отдельные границы, паддинг и маржин. Примеры блочных элементов: div, p, h1, form.
2. Строчными элементами называются элементы, которые занимают только необходимое место и не имеют отдельных границ, паддинга и маржина. Примеры строчных элементов: span, strong, em, a.
3. Задать свойства блочных элементов строчным можно с помощью свойств display: inline-block;
4. Сокращенная запись свойства margin включает в себя четыре значения: margin-top, margin-right, margin-bottom и margin-left.
5. Свойство box-shadow принимает следующие значения: горизонтальное смещение, вертикальное смещение, радиус размытия, цвет и признак тени (inset).
6. Для размещения элемента по центру можно использовать свойства margin или text-align.
7. Блочные элементы имеют следующие свойства: ширина, высота, отступы, паддинги, маржины, границы, позиционирование и отображение.
8. Свойства границы блочных элементов включают в себя цвет, толщину, стиль и радиус скругления.
9. Ширина блочного элемента определяется с помощью свойства width.
10. Свойство box-sizing используется для задания размера блочного элемента включая границу и паддинг.
11. Свойство overflow используется для указания браузеру, как он должен поступить с элементами, которые не помещаются в доступное пространство.
12. Значения в данном объявлении обозначают цвета границы следующим образом: yellow - верхняя граница, red - правая граница, green - нижняя граница и blue - левая граница.
13. Для задания ширины только правой границы можно использовать свойство border-right-width.
14. Для создания элемента с 5px пунктирной границей красного цвета с отступом от текста сверху 20px можно использовать следующее объявление:

element { margin-top: 20px; border: 5px dotted red; }

15. Свойство border-style принимает следующие значения: none, hidden, dotted, dashed, solid, double, groove, ridge, inset, outset.
