1. Селектор атрибутов - это синтаксис, который позволяет идентифицировать HTML-элементы по их атрибутам и составляющим.
2. Часто используемые селекторы атрибутов: [id], [class], [name], [title], [lang], [type], [data-*].
3. Для этого можно использовать свойство border-collapse: collapse.
4. Для удаления пустых ячеек можно использовать свойство empty-cells: hide.
5. Значения в сокращенной записи свойства padding означают отступы в пикселях по отношению к границам элемента. Эти значения могут быть заданы для всех границ сразу, для каждой границы по отдельности или для горизонтальных и вертикальных границ сразу.
6. Псевдокласс focus используется для стилизации элементов, когда они имеют фокус ввода.
7. Псевдокласс nth-child(odd) используется для определения нечетных элементов списка.
8. Псевдокласс nth-child(even) используется для определения четных элементов списка.
9. Для добавления тени к элементам формы можно использовать свойство box-shadow.
10. Для изменения промежутка между ячейками используется свойство cellspacing.
11. Свойство border-radius используется для создания закругленных границ элементов.
12. Для выравнивания текста в таблице по вертикали используется свойство vertical-align.
13. Для выравнивания текста в таблице по горизонтали используется свойство text-align.
14. Значения, которые может принимать свойство vertical-align: baseline, length, sub, super, top, bottom, middle, text-top, text-bottom. 
15. Псевдокласс first-child используется для поиска первого элемента в списке.
16. Псевдокласс last-child используется для поиска последнего элемента в списке.
17. Теги <thead>, <tfoot> и <tbody> используются для разделения таблицы на секции. <thead> используется для хранения шапки таблицы, <tfoot> - для хранения подвала таблицы, а <tbody> - для хранения основного контента таблицы.
18. Свойство border-spacing и его значение separate используется для определения промежутка между ячейками таблицы.
19. Свойство border-collapse и его значение collapse используется для объединения границ таблицы.
20. Тег <colgroup> используется для определения группы столбцов, а тег <col> предназначен для задания атрибутов столбцов таблицы.
21. Значение baseline означает линию, которая отображается по нижней границе текста, изображений и других элементов.
