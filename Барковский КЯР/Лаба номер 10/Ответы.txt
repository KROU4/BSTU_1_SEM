1. Фиксированная верстка веб-страницы это способ дизайна, при котором размеры веб-страницы не изменяются при изменении размеров окна браузера или при отображении на устройствах с разными разрешениями экрана.

2. Резиновый макет веб-страницы это способ дизайна, при котором размеры веб-страницы изменяются в зависимости от размеров окна браузера или разрешения экрана, а также изменяются пропорционально к изменениям в размерах.

3. Адаптивный дизайн - это метод дизайна, при котором веб-страница адаптируется к различным устройствам и их размерам экрана, а также использует разные макеты для разных устройств.

4. Медиазапросы - это инструмент для определения особенностей устройства и изменения стилей веб-страницы в зависимости от этих особенностей. Медиазапросы можно подключить, используя правило @media в CSS-файле.

5. Правило @import используется для импортирования других файлов CSS в данный файл.

6. Правило @media используется для определения стилей в зависимости от особенностей устройства, а также для определения стилей для различных размеров окна браузера.

7. Свойство flex используется для управления размерами и положением элементов внутри flex-контейнера.

8. @media (min-width: 560рх) означает, что стили будут применяться для устройств с шириной окна браузера не менее 560px.

9. @media (max-width: 960рх) означает, что стили будут применяться для устройств с шириной окна браузера не более 960px.

10. flex: 1 2 200px; означает, что элемент будет иметь ширину 200px, а прочие параметры (высота и плавающие параметры) задаются автоматически.

11. Значение wrap предназначено для того, чтобы элементы flex-контейнера переносились на следующую строку при изменении размеров окна или экрана.

12. Свойство flex-flow используется для указания направления потока элементов в flex-контейнере.

13. Свойство flex-direction используется для указания направления элементов в flex-контейнере.

14. Свойство align-items и свойство align-content используются для выравнивания элементов flex-контейнера по горизонтали и вертикали соответственно.

15. Свойство align-self имеет три значения: auto, flex-start и flex-end. Оно используется для выравнивания отдельных элементов flex-контейнера.

16. Flex-контейнер имеет следующие свойства: display, flex-direction, flex-wrap, flex-flow, justify-content, align-items и align-content.

17. Flex-элементы имеют следующие свойства: flex, flex-grow, flex-shrink, flex-basis, align-self.

18. Свойство justify-content используется для выравнивания элементов flex-контейнера по горизонтали.

19. Видео можно вставить, используя тег <video>, а аудио - с помощью тега <audio>.

20. Свойство align-items устанавливает выравнивание для всех элементов flex-контейнера, а свойство align-self устанавливает выравнивание для отдельных элементов.

21. Адаптивное меню можно создать, используя медиазапросы и стилизацию элементов.

22. @media (max-width: 768px) {
    p {
        color: red;
    }
}

23. Специфичность селектора определяет, насколько точно он относится к конкретному элементу на веб-странице. Значения специфичности могут быть нулевыми, положительными или отрицательными. Нулевое значение специфичности является наименее специфичным, тогда как положительное и отрицательное значения более специфичны.
