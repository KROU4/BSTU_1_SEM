методы перебора массива



1. for loop
2. forEach()
3. for...in
4. for...of
5. map()
6. filter()
7. reduce()
8. find()
9. some()
10. every()





for loop: Этот метод предназначен для последовательного итерирования по массиву и выполнения действий для каждого элемента. Он предоставляет пользователю полную гибкость в управлении циклом, поскольку позволяет задать условие для выхода из цикла и изменять переменные счетчика.

forEach(): Этот метод предназначен для итерации по массиву и выполнения для каждого элемента заданной функции. Он автоматически устанавливает индекс для каждого элемента и передает его в функцию, поэтому пользователь не должен беспокоиться об установлении индекса и его изменении.

for...in: Этот метод предназначен для последовательной итерации по индексам (ключам) массива. В качестве аргумента он принимает индекс и при каждой итерации выполняет заданное действие для данного индекса.

for...of: Этот метод предназначен для последовательной итерации по значениям массива. Он принимает значение и при каждой итерации выполняет заданное действие для данного значения.

map(): Этот метод предназначен для последовательной итерации по массиву и преобразования каждого значения в другое значение, используя заданную функцию. Он автоматически устанавливает индекс для каждого элемента и передает его в функцию, поэтому пользователь не должен беспокоиться об установлении индекса и его изменении.

filter(): Этот метод предназначен для последовательной итерации по массиву и выбора определенных элементов из него. Для этого используется заданная функция, которая принимает значение и возвращает true или false в зависимости от того, соответствует ли данное значение условию. Все значения, для которых функция вернула true, будут добавлены в возвращаемый массив.

reduce(): Этот метод предназначен для последовательной итерации по массиву и преобразования всех значений массива в одно значение с помощью заданной функции. Он автоматически устанавливает индекс для каждого элемента и передает его в функцию, поэтому пользователь не должен беспокоиться об установлении индекса и его изменении.

find(): Этот метод предназначен для поиска значения в массиве. Для этого используется заданная функция, которая принимает значение и возвращает true или false в зависимости от того, соответствует ли данное значение условию. Все значения, для которых функция вернула true, будут добавлены в возвращаемое значение.

some(): Этот метод предназначен для проверки массива на то, содержит ли он по крайней мере одно значение, которое удовлетворяет заданному условию. Для этого используется заданная функция, которая принимает значение и возвращает true или false в зависимости от того, соответствует ли данное значение условию. Если хотя бы одно значение возвращает true, то метод some вернет true, в противном случае он вернет false.

every(): Этот метод предназначен для проверки массива на то, все ли его значения удовлетворяют заданному условию. Для этого используется заданная функция, которая принимает значение и возвращает true или false в зависимости от того, соответствует ли данное значение условию. Если все значения массива удовлетворяют функции, то метод every() возвращает true. В противном случае он возвращает false.
